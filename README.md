# Parking_Fee
A Matlab function that calculates the cost of a (hypothetical) parking ticket, dependent on time spent at the lot.
## Input Explanations

Time Spent | Cost
-----------|-----
(up to) 2 hours    | $4
4 hours    | $7
Each Additional Hour | + $1
24 hours   | $24
Lost Ticket | $36

A lost ticket is symbolized in the input by a negative integer
